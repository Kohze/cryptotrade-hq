Slack Bots can help share information and improve the communication between community members. We created a bot that takes trading movements of our members via bitfinex API and shares in our bot channel daily position movements.

FAQ

While the slack bot is still under development, we can already give some answers.

  • What is it about

    Our slack trading movement bot tracks the bitfinex trades of all our members that submitted their API key to participate. It allows slack members to follow actual trading movements rather than just trusting arguments.

  • Is it safe?

    Since we only need the "wallet view" API key from bitfinex and no trade access at all, those information are not only anonymized but also completly safe to share.

  • How do I participate

    You can submit your bitfinex API key in the form below. Make sure to only activate wallet view permissions before you submit the form. If our algorithmic check cannot access the data or finds, that to many rights were granted, the key gets automaticly deleted.

  • How do I delete my API key from the bot again?

    In Bitfinex, you can delete and recreate a API key at any time. Just delete the shared API key and the bot will automatically unfollow your trading movements.
